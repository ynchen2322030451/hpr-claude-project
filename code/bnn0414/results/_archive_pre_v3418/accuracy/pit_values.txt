================================================================================
pit_values.npz — PIT 原始值（Fig. 1B/C 数据源）
================================================================================

【秒懂版】
一句话：所有模型的概率积分变换（PIT）值——画PIT直方图的原始数据。

通俗解释：
npz 文件，每个 key 是 model_id，值是 (N_test, N_outputs) 的数组。
PIT值应该均匀分布在[0,1]之间，否则说明模型校准有偏。
用 np.load(path) 加载。

【详细版】
论文定位：Fig. 1B/C 数据源
格式：np.load(path), keys = model_id, values = ndarray (435, 15)
