# 在 PowerPoint 中手动把 2D 剖面图拉伸成立体效果

## 总体思路

立体反应堆俯视图 = **二维剖面图** + **3D 旋转倾斜** + **向下拉伸出厚度**。
PowerPoint 自带的「三维格式」与「三维旋转」可以直接完成这两步，无需任何建模软件。

> **前置文件**：`core_section_2d.pptx`（纯 2D 剖面，无 3D 效果）
> **参考结果**：`core_cross_section_3d.png`（Python 已经渲染好的立体效果）

---

## 第一步：把 2D 图拉伸成「厚饼」

1. 打开 `core_section_2d.pptx`。
2. 选中整张图片（左图：全剖面圆盘；右图：1/6 扇形）。
3. 菜单栏 → **格式 / 图片格式** → 最右侧 **图片效果** → **预设** → 最下面 **三维选项（3-D Options…）**。
   （或右键 → **设置图片格式 / Format Shape** → 点 **效果** 图标 → 展开 **三维格式 3-D Format** 与 **三维旋转 3-D Rotation**。）

4. 在 **三维格式 / 3-D Format** 下：
   - **深度 Depth**：`36 pt`（推荐 30–60 pt，数值越大越像"厚圆柱"；太大会遮挡顶面细节）
   - **深度颜色 Depth Color**：手动选一个比顶面稍暗的金色（比如 RGB `180,135,25`）——这样侧壁有立体感。
   - **顶部棱角 Top Bevel / 底部棱角 Bottom Bevel**：留空（不要加圆角，否则会破坏剖面边界的锐利感）。
   - **表面 Surface**：
     - **材质 Material**：`柔边 Soft Edge` 或 `哑光 Matte`（避免金属高光）
     - **照明 Lighting**：`中性 / Neutral`（柔和）→ 方向 45°
   - **曲面角度 Contour**：0 pt（不要等高线）

5. 在 **三维旋转 / 3-D Rotation** 下：
   - **预设 Presets** → **等角轴测：右上 Isometric Top Up**（或手动调参数）
   - 或手动输入：
     - **X 旋转**：`0°`
     - **Y 旋转**：`0°`
     - **Z 旋转**：`0°`（如果想让切口朝向某个方向，调整 Z）
     - **透视 Perspective**：`0°`（保持正交投影，避免近大远小）
   - 如果要接近 Python 版：X=15°，Y=0°，Z=215°，透视 0°。数字可按喜好微调。

---

## 第二步：如果要让切口（扇形口）更显"立体"

单张图片拉伸后，**切口处的侧壁会和周围圆柱一起拉伸**，所以切口看起来就是"被切掉一块"。
若想让切面看起来更明显（比如想显示轴向分层），可用以下两种方法：

### 方法 A：两张图叠层
- 第一层：完整圆盘（仅顶面），拉伸 `40 pt`，浅金色。
- 第二层：把 `core_section_2d.pptx` 左图中的**反射体扇环部分**复制一份，裁剪成 **120° 的弧形**，不拉伸（Depth=0），平放在下层之上，用它遮住切口处的"底面"——即可造出剖切面被"切出来"的错觉。

### 方法 B：剖面图贴到切面上
- 额外准备一张**轴向剖面矩形图**（气体室→反射体→燃料→上反射体，垂直分层）。
- 旋转到与切口平面对齐的角度，再用"裁剪为形状"把它贴到切口位置。
- 这是 PPT 里的"贴图"方法，常用于复杂堆芯拉伸示意。

---

## 第三步：微调光影以获得论文插图质量

1. 选中拉伸对象 → **三维格式 → 照明**：把「预设光源」设为 **3 点光源** 或 **柔光 Soft**。
2. **光照角度**：45° 使金色侧壁比顶面暗约 15%，人眼看上去就会识别为"上亮下暗"的立体感。
3. **阴影 Shadow**：
   - 类型：偏移下方 **Offset: Bottom**
   - 模糊：`6 pt`
   - 距离：`4 pt`
   - 颜色：深灰 60% 透明
   - 这样圆盘下会投一圈软阴影，进一步强化立体。
4. 导出：**文件 → 另存为 → PNG/PDF**，分辨率建议 300 dpi。

## 常见问题

| 现象 | 原因 | 处理 |
|------|------|------|
| 顶面细节（堆芯 pin）模糊 | 拉伸 Depth 太深 | 减到 20–30 pt |
| 侧壁出现"透视变形" | 透视 Perspective > 0 | 置为 0° |
| 切口看起来是空洞 | 没有底面遮挡 | 叠一张同形状低位图片做底板 |
| 色带出现锯齿 | PNG 被压缩 | 用 SVG/PDF 导出 |
| 控制鼓绿色弧被裁掉 | 顶面裁剪路径设错 | 重新设置剪切区域或改用矢量 SVG |

## 小结

- 真正让图"立体"的只是**深度 + 旋转**这两步；
- Python 生成的 `core_cross_section_3d.*` 本质就是数学地做了这件事；
- 对 PPT 用户，从 `core_section_2d.pptx` 出发，3 分钟就能得到类似 Python 版的立体图；
- 如果后期要换配色 / 堆芯参数 / 切角比例，只需改 Python 脚本里的颜色常量 + 切角度数，再按同样流程做一遍 PPT 拉伸即可。
