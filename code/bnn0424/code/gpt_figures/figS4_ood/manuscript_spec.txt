【秒懂版】
一句话：当输入参数偏离训练范围时，BNN的不确定性确实会增大——它知道自己"不确定"。

通俗解释：
一个好的概率模型应该在不熟悉的区域表达更大的不确定性。这张图测试了4个参数
分别超出训练范围尾部时，BNN的epistemic标准差是否增大。所有柱子都 >1
（7-21%增长），说明BNN确实在OOD区域更不确定。虚线=1是分布内基线。
